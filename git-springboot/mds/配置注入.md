1、注入方式
    ```
    @Value 用于单属性注入
    @ConfigurationProperties(prefix = "user") 用于对象型注入
    ```
2、配置值
    
```yaml
user:
    hobbies: ['游戏','运动']
    birthday: 2020/01/01
    hobbies2:
      - 美女
      - 游戏
```
    备注：
    **日期类型：2020/01/01**
    **支持${random.uuid}**
    **yaml提示需要导入spring-boot-configuration-processor**
